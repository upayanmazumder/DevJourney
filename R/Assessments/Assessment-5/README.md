# Assessment 5 - Solutions

This folder contains per-question R Markdown solutions for Assessment 5.

Files and folders:

- `Q1/Question1.Rmd` - Latin Square Design ANOVA (wheat varieties)
- `Q2/Question2.Rmd` - One-way ANOVA (filling machine efficiency)
- `Q3/Question3.Rmd` - One-way ANOVA (training programs)
- `Q4/Question4.Rmd` - Worker efficiency analysis with block and machine effects

How to run:

1. Open the desired `QuestionX.Rmd` file.
2. Knit to HTML in RStudio or run `rmarkdown::render("QX/QuestionX.Rmd")`.
