# Assessment 4 - Solutions

This folder contains per-question R Markdown solutions for Assessment 4.

Files and folders:

- `Q1/Question1.Rmd` - Chi-square test of independence
- `Q2/Question2.Rmd` - Binomial fitting and goodness-of-fit test
- `Q3/Question3.Rmd` - Paired t-test (regular vs premium fuel)
- `Q4/Question4.Rmd` - Paired t-test (training effectiveness)
- `Q5/Question5.Rmd` - Paired t-test (IQ before vs after training)

How to run:

1. Open the desired `QuestionX.Rmd` file.
2. Knit to HTML in RStudio or run `rmarkdown::render("QX/QuestionX.Rmd")`.
