Algorithms Lab — C++ Implementations (LCP)

This folder contains study-ready explanations and clear C++ implementations for ten classic algorithms used in the LCP / Algorithms lab. Each subfolder includes:

- a concise problem statement
- the main idea and a short pseudocode sketch
- complexity analysis and practical notes
- a worked example you can trace by hand
- the reference implementation in `main.cpp`

How to compile and run

- Build: `g++ -std=c++17 -O2 main.cpp -o main.exe`
- Run (Windows PowerShell): `.\\main.exe`
- Run (Linux/macOS): `./main.exe`

Study tips

- Read the problem statement, then the algorithm idea, then step through the worked example by hand.
- Compare the pseudocode to the implementation in `main.cpp` to connect theory with code.
